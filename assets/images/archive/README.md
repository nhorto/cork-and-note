# Archived images

Retired assets kept for reference. Nothing here is referenced by the app.

## `VCN_red.jpg`

The original login-screen logo — a dark-burgundy square reading **"VIRGINIA
CORK & NOTE"**. Retired in the de-Virginia rebrand (issue #38), replaced on the
login screen by `assets/images/cork_and_note_logo.png`.

To restore it, point the `<Image>` in `app/login.js` back at this file:

```js
source={require('../assets/images/archive/VCN_red.jpg')}
```
